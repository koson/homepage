OpenEnergyMonitor System

Pre-assembled open hardware electricity, temperature and humidity monitoring units based on and compatible with the Arduino and RaspberryPi platforms.

EmonTx
4x CT Inputs
1x ACAC Adapter
1x Pulse input
DS18B20 based temperature sensing
EmonTH
Temperature and Humidity node
Pulse counting
Battery operated
EmonPi
2x CT Inputs
1x ACAC Adapter
1x Pulse input
DS18B20 based temperature sensing
EmonCMS
Open source energy data processing, storage and visualisation.
Hosted service or self hostable web application.
